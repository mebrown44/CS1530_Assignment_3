The Only Cans

SteelHacks XI Winner: 1st in Sustainability, 1st in Civil Design, 3rd in Data Observability

Allows users to rate trash cans and recycling bins in the community based on fullness in order to optimize the trash collection process. It implements a Leaflet.js map feature to map trash cans and find the closest one to you. It is coded in HTML and Javascript and is hosted using Github Pages.
Website: https://theonlycans.co/
Github Repository: https://github.com/mebrown44/Only-Cans-Final

Handimapped

SteelHacks XII Project

Users can register buildings to the database and rate its accessibility and explain why to help disabled individuals plan for their day. Users can view previous ratings on a Leaflet.js map feature and read the reviews that users leave. This project was created using HTML, Javascript, MySQL + Aiven Database Software, Vercel website hosting.
Website: https://handimapped.com/
Github Repository: https://github.com/mebrown44/HandiMapped

Gateway Research: Prehispanic Oaxacan City Database

Research with the University of Pittsburgh Department of Anthropology, part of URBank

Focusing on prehispanic cities in the modern Oaxaca state of Mexico, we reviewed past research and literature to compile a database of location, size, chronology, and population data. The purpose of this is to help future researchers and resolve conflicting accounts. This project is a part of the larger URBank Urbanization project with the Max Planck Institute of Geoanthropology and other academic institutions. In addition to the literature reviews, GIS components were completed on ArcGIS.
URBank Website: https://urbank.earth/
